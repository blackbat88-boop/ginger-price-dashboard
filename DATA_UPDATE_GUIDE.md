# 生姜价格数据更新指南

## 文件说明

`data.js` - 存储所有生姜价格数据，修改此文件即可更新网站内容。

## 更新步骤

### 1. 每日更新（推荐频率：每天或每 2-3 天）

打开 `data.js`，找到 `history` 数组：

```javascript
history: [
    ['3/30', 3.51, 5.18],
    ['3/31', 3.52, 5.20],  // ← 最新一天
    ['4/1', 3.53, 5.22],   // ← 添加新行
]
```

**格式说明**：`['月/日', 全国均价，沂水价格]`

### 2. 更新产区价格（可选）

在 `regions` 对象中更新各产区价格：

```javascript
shandong: [
    { name: '沂水（小黄姜）', price: 5.00, change: 0.28 },
    // change: 涨跌幅度（百分比），上涨为正数，下跌为负数，持平为 0
]
```

### 3. 更新预测数据（可选，每周更新）

在 `forecast` 对象中更新预测：

```javascript
7: {
    national: [3.53, 3.55, 3.57, ...],  // 未来 7 天全国预测
    yishui: [5.22, 5.25, 5.28, ...]     // 未来 7 天沂水预测
}
```

### 4. 更新市场分析（可选）

```javascript
marketAnalysis: {
    dynamic: '当前市场动态描述...',
    forecast: '价格预测分析...'
}
```

## 数据来源建议

1. **我的钢铁网** - https://www.mysteel.com/
2. **中姜网** - 微信公众号/行业群
3. **产区经纪人** - 微信/电话报价

## 发布更新

1. 修改 `data.js`
2. 保存文件
3. `git add data.js && git commit -m "update: 更新生姜价格至 MM 月 DD 日"`
4. `git push`
5. GitHub Pages 会自动刷新（约 1-2 分钟）

## 注意事项

- 价格单位：**元/斤**
- 涨跌幅计算：`(今日价 - 昨日价) / 昨日价 * 100`
- 保持 `history` 数组至少有 30 条记录（用于 30 天图表）
- 预测数据仅供参考，建议基于市场经验填写
