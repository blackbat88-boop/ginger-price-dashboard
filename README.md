# 生姜价格监控中心

实时追踪山东主产区生姜价格动态，包含30天走势图、价格预测和农户销售建议。

## 访问地址

https://你的用户名.github.io/ginger-price-dashboard

## 功能特性

- 📈 30天价格走势图（全国均价 + 沂水地区）
- 🔮 未来价格预测（7天/30天/90天）
- 💡 农户销售建议
- 🏆 全国主产区价格排行榜
- 📋 标准化数据采集规范

## 数据来源

- 我的钢铁网 Mysteel
- 中姜网
