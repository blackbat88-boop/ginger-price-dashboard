## 🔄 自动更新

所有页面每天 **北京时间 10:00** 自动更新价格数据：

1. 从惠农网抓取最新价格
2. 解析各地区价格数据
3. 更新 HTML 页面
4. 自动推送到 GitHub
5. GitHub Pages 自动部署

## 📝 数据说明

- **价格类型**：产区主流成交价（非零售价）
- **计量单位**：元/斤（500克）
- **质量标准**：优质、主流、次货分级
- **更新时间**：每日上午 9:00-11:00 采集
- **数据来源**：惠农网、Mysteel、中姜网
